Featured Projects

This is a website that allows users to add their own reviews for their favorite anime. Their reviews get saved to a database and displayed for other people to see. If a user wishes to edit or delete their review, they have the ability to do just that. There is also a section for users to search for anime using a 3rd Party API. The search results are then displayed for the user to see. Clicking on one of the results will take them to the anime's page on My Anime List. Lastly, the is a section for users to catch up on the latest anime news. This is accomplished by using Beautiful Soup to scrape data from the My Anime List news section. The user can click on a news article, and be taken to that article's page on My Anime List.
I was tasked with designing aand implementing a number of features. The first task was to style the Home Page per the clients request. This was done using simple HTML and CSS. Next, I was to create an entity model for calendar events that could be scheduled, and create the CRUD pages for it. Entity Framework made this a very simple task. After that, I had to link the calendar events to the production class. This was accomplished with a one-to-many relationship between the two classes where one production can have many calendar events. Then I was tasked with redesigning and styling the rental requests page. The previous design had the data displayed in a simple table. My job was to have the data display in an accordion. This one was a bit of a challenge, but was eventually overcome by some quick research. Lastly, I had to implement a way to only show current rental requests until the user clicked a button to show the expired requests. I had a tough time with this one, but like any other challenge it was quickly overcome. The solution was just a matter of using jQuery to handle the button function.
